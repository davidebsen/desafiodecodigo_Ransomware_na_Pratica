# Desafio de codigo.

Este repositório contém dois scripts em Python para criptografar e descriptografar.

## Arquivos

1. **`encrypter.py`**: Script para criptografar.
2. **`decrypter.py`**: Script para descriptografar.
3. **`desafiodio.txt`**: Arquivo de exemplo para ser criptografado e descriptografado.
